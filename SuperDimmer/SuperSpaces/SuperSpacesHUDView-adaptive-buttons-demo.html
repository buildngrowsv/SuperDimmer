How Adaptive Sizing Works

  • Compact Mode (< 400px): Shows only emoji or number - minimal space usage
  • Medium Mode (400-550px): Shows number + emoji - balanced view
  • Expanded Mode (> 550px): Shows number + emoji + name - full information
  • Smooth Transitions: Buttons smoothly transition between modes as you resize
  • Note Indicators: Orange dots show which spaces have notes (always visible)
